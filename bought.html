Instructions

  1. Copy the file to your versions folder inside .minecraft. Its directory should be like C:\Users\Yourname\AppData\.minecraft\versions\Wrust-1.13.jar.
  2. Open up cmd.
  3. Type in "cd AppData" without quotes.
  4. Type in "cd .minecraft" without quotes.
  5. Type in "cd versions" without quotes.
  6. Type in "java -jar Wrust-1.13.jar"
  7. The client will install itself now. Enjoy!
